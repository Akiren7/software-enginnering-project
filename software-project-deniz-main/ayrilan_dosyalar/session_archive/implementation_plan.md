# Simple Python Server-Client (HTTP + WebSocket)

A basic server that handles both HTTP requests and WebSocket connections, with a matching client.

## Project Structure

```
6064/
├── requirements.txt          # aiohttp
├── server.py                 # HTTP routes + WebSocket endpoint
├── client.py                 # Demo client (HTTP calls + WS chat)
└── shared.py                 # Shared JSON message helpers
```
